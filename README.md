# Python Your Virtual Environment

## Linux / Mac

```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Windows - Command Prompt

```
python -m venv env
env\Scripts\activate.bat
pip install -r requirements.txt
```

## Windows - Power Shell

As Administrator:
```
Set-ExecutionPolicy Unrestricted
```

As user:
```
python -m venv env
env\Scripts\activate.ps1
pip install -r requirements.txt
```
